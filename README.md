# 🛒 NeoCart - Modern E-Commerce Website
NeoCart is a fully responsive modern e-commerce frontend website built using HTML, CSS, JavaScript, Bootstrap, jQuery, and LocalStorage.
This project was developed as part of an internship project requirement for designing a complete e-commerce website with modern UI and ecommerce functionalities.
---
# 🚀 Features
## 🏠 Home Page
- Responsive navbar
- Hero section
- Shop by category
- Latest products
- Features section
- Footer
- Scroll-to-top button
## 🛍 Products Page
- Product listing
- Search products
- Category filtering
- Price range filtering
## 📦 Product View Page
- Product gallery
- Ratings
- Quantity controls
- Add to cart
- Related products
## 🛒 Cart System
- Add/remove products
- Quantity management
- Dynamic total calculation
- Persistent cart using LocalStorage
## 💳 Checkout System
- Shipping information form
- Billing details
- Order summary
- jQuery form validation
## ✅ Order Confirmation
- Order success page
- Dynamic order ID
- Order status
## 📂 Additional Pages
- Category page
- About Us page
- Contact Us page
- Search page
